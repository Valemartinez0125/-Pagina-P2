6. ¿Qué es la factorización?
ç
R/ La factorización de un proceso de expresar un numero a una expresión algebraica como el producto de dos o mas factores. En otras palabras, consiste en descomponer un numero o expresión en sus componentes básicos, que se multiplican entre sí, para obtener el original.
7. Factorización por factor común.
R/ Es un método utilizado para factorizar expresiones algebraicas que tiene dos o mas términos que comparten un factor común. El procedimiento es el siguiente:
*Identificar el factor común: Buscar el factor que es común y todos los términos de la expresión.
*Factorizar: Escribir la expresión con el producto del factor común y una expresión que contiene términos restantes.
8. Factor común por agrupación de términos.
R/ La factorización por agrupación de términos es un método utilizado para factorizar expresiones algebraicas que tiene cuatro o más términos. Este método implica agrupar los términos en pares y buscar factores comunes dentro de cada par. El procedimiento es el siguiente:
*Agrupar los términos: Dividir la expresión en pares de términos.
*Factorizar cada par: Buscar factores comunes dentro de términos.
*Factorizar el resultado: Buscar factores comunes entre los resultados de cada par y factorizarlos.
9. Diferencia de cuadrado
R/ La diferencia de cuadrado es una identidad algebraica que se utiliza para factorizar expresiones que contiene las diferencias de dos cuadrados.
10. Trinomio cuadrado perfecto
R/ Un trinomio cuadrado perfecto es una expresión algebraica que se puede factorizar como el cuadrado de un binomio.
11. Trinomio de la forma X 2 + b x + c
R/ Es una expresión algebraica que contiene 3 términos: Un término cuadrático (X 2 ), un termino lineal (b x) y un término constante (c). La forma general de un trinomio es X 2 + b x + c, donde: *X es la variable * b es el coeficiente del término literal * c es el término constante
12. Trinomio de la forma a x 2 + b x + c con a=/ 1
R/ Es una expresión algebraica que contiene tres términos. Para factorizar un trinomio de esta forma, se pueden utilizar varios métodos como: 1 factorización por agrupación 2 Factorización por búsqueda de factores comunes
13. Suma o diferencia de cubos perfectos
R/ La suma o diferencia de cubos perfectos se refiere a la factorización de expresiones que contienen la suma o diferencia de dos cubos perfectos.
